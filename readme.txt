Adds a boat customizer (builder) backend and frontend.

== Description ==
Adds a boat custom post type for creating boats with specified parts and pricing. It allows for detailed conditions and logic, email, and PDF generation.

= Dependencies =
* Uses [Plugin Update Checker](https://github.com/YahnisElsts/plugin-update-checker) for providing updates from GitHub

= Browser Support =

== Installation ==
1. Download .zip
2. Import .zip into plugins
3. Activate plugin
4. Create your boats using the Boat CPT
